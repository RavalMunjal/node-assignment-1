## 📌 Objective

Build a REST API using **Express.js** that manages student CGPA records stored in an in-memory JSON array.

This project:

- Uses only GET routes
- Implements static and dynamic routes
- Follows REST principles
- Returns proper HTTP status codes
- Uses an in-memory JSON array (no database)

---

## 🛠 Tech Stack

- Node.js
- Express.js
- CORS
- In-memory JSON array
